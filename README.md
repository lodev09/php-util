PHP Util
============================

Collection of methods that I or you usually use.

## Installation
```term
$ composer require lodev09/php-util
```

## Usage
```php
// for example, slugify a string like so
$slugified = \Common\Util::slugify('slugify me');
echo $slugified;
```
